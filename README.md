# Python Electricity Bill Calculator

A beginner-friendly Python project that calculates electricity bills using slab-based pricing.

## Features
- Takes electricity units as input
- Calculates bill using slab rates
- Uses conditional statements
- Simple console-based program

## Technologies Used
- Python 3

## Slab Rates
- First 50 units → ₹2 per unit
- Next 50 units → ₹3 per unit
- Above 100 units → ₹5 per unit

## How It Works
The user enters the number of electricity units consumed, and the program calculates the total bill amount based on slab pricing.

## Concepts Used
- if-elif-else conditions
- Arithmetic operations
- User input handling

## Run the Program

```bash
python electricity_bill.py
